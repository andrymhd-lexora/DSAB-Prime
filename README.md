# DSAB PRIME • Portofolio Konstruksi Sipil Terintegrasi

**DSAB Prime** (PT DSAB Indonesia) adalah portfolio digital interaktif untuk jasa konstruksi sipil terintegrasi yang berfokus pada pembangunan infrastruktur, gedung vertikal komersial, restorasi sejarah, dan konstruksi hijau ramah lingkungan di seluruh Indonesia.

Website ini mengusung estetika **Dark Futuristic** sasis ungu-sian berenergi tinggi, mendukung penuh interaktivitas real-time, bilingualisme (Bahasa Indonesia / English), serta dilengkapi simulasor estimasi anggaran (CapEx) konstruksi sipil nasional.

---

## 🌌 Fitur Unggulan Sistem

1. **Bilingual Seamless Support**: Sistem transfer bahasa menyeluruh via satu tombol di header tanpa refresh halaman.
2. **Standard & Premium Dark/Light Modes**: Toggle visual komprehensif yang melunakkan pancaran cahaya (eye-safe) menjadi mode korporat profesional yang jernih.
3. **Interactive Budget Estimator**: Kalkulator biaya cerdas berskala meter persegi/linear, terbiasakan dengan koefisien material standar sipil nasional (K-350, K-500, Carbon) serta indeks multiplier kewilayahan (Jabodetabek, IKN, Surabaya, dsb).
4. **WhatsApp Spec Transmit Generator**: Tombol otomatisasi yang menyusun rancangan spesifikasi kalkulator menjadi tautan draf konsultasi instan ke divisi penawaran kami.
5. **Dynamic Virtual Telemetry Feed**: Panel logging real-time di bagian bawah yang mementaskan transparansi diagnostik dari aksi pengguna layaknya dashboard kontrol pusat K3 (Kesehatan & Keselamatan Kerja) modern.
6. **Specifications Inspector Modal**: Pop-up interaktif detail proyek lengkap dengan navigasi melingkar (Prev/Next Carousel) guna mengeksplorasi spesifikasi beton, struktural, serta sertifikasi hijau.
7. **Loading Skeleton Screen**: Simulasi transisi pemuatan (loading metrics animatif) sewaktu menyaring kategori portofolio untuk menambah kenyamanan pengguna.

## 🛠️ Tech Stack & Konfigurasi

- **UI Framework**: React 19 dengan TypeScript kelonggaran tipe bundler
- **Styling Guide**: Tailwind CSS v4 dengan Google Fonts (Space Grotesk & Plus Jakarta Sans)
- **Fluid Animations**: Motion (`motion/react`)
- **Icons**: Lucide React
- **Dev Servers**: Vite 6, tsc, dan Express

## 🚀 Panduan Instalasi & Pengembangan Lokalan

### 1. Prasyarat
Pastikan mesin pengembang Anda memiliki Node.js v18+.

### 2. Kloning & Menginstal Dependensi
Pasang semua pustaka yang terdaftar pada `package.json`:
```bash
npm install
```

### 3. Menjalankan Kode di Mode Pengembangan
Luncurkan server pengembangan lokal Vite yang mengarah langsung ke port 3000:
```bash
npm run dev
```
Buka peramban Anda di alamat: `http://localhost:3000`.

### 4. Membangun Kode untuk Produksi
Guna melakukan kompilasi bundel production-ready ke direktori `dist/`:
```bash
npm run build
```

### 5. Memeriksa Integritas Tipe Linting
Jalankan pendaftar statis TypeScript tanpa emisi berkas:
```bash
npm run lint
```
